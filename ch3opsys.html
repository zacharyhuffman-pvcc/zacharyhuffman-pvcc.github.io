Operating System Terminology
Term Definition
Application Software Consists of the programs for performing tasks particular to the intended function.
Boot Strapping Performed by a computer each time it is turned on; it transfers the operating system from mass storage into main memory.
Deadlock The condition in which two or more processes are blocked from progressing because each is waiting for a resource allocated to another.
Device Driver Software units that communicate with controllers or peripheral devices to carry out operations on the attached devices.
File Manager Mantains records of all files stored in mass storage, including their locations, user permissions, and available storage space.
FIFO A processing system that works on jobs based on the order they are received in the job queue.
Graphical User Interface (GUI) An interactive interface where users interact with shapes, folders, and icons instead of using command lines.
Interactive Processing The process where a user provides input and the machine returns data for further user response.
Kernel The core of the operating system that manages system files and programs.
Memory Manager A part of the kernel that coordinates the allocation of main memory for system tasks.
Multi-factor Authentication A security process that requires multiple forms of verification before granting access.
Multiprogramming A method where time intervals are set for tasks, allowing the system to switch between jobs, creating the illusion of simultaneous processing.
Operating System The software that controls the overall operation of a computer and provides an environment for executing programs.
Shell An older user interface where users communicate via keyboard and textual inputs.
Spooling The process of a printer staggering job orders until earlier jobs in the queue have been processed.
Super User (or administrator) The highest privileged user on an operating system with the ability to change crucial settings and permissions.
System Software The operating infrastructure needed for applications to function correctly; it acts as the highways and laws for programs.
Time-Sharing A method that allows multiple users to interact with resources simultaneously.
Utility Software Software that extends the capabilities of the operating system to perform essential tasks like formatting and installation.
Virtual Memory The temporary use of main memory or hard drive space as RAM when processing multiple tasks or memory-intensive processes.
