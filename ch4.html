Chapter 4: Networking

Zac Huffman

Terms Definitions
Network Classifications
LAN (Local Area Network) A geographically local computer network usually located within a single building made up of connected computers, printers, and equipment.
MAN (Metropolitan Area Network) Geographically larger than a LAN, usually covering a city and many buildings by connecting multiple LANs.
WAN (Wide Area Network) A massive network covering a country or the globe, connecting multiple LANs and MANs together.
Network Models
Bus Network network model where all devices are connected to a central cable or "highway" that carries all network data.
Star Network A network model where all devices are connected to a central hub or switch. (In a more interactive way, airlines use a version of this for routing airline traffic)
Protocols
CSMA/CD Ethernet Carrier Sense, Multiple Access with Collision Detection is a protocol that dictates which machine in a bus network topology has the right to transmit data at a given time to prevent collisions
CSMA/CA Wi-Fi Carrier Sense, Multiple Access with Collision Avoidance is a protocol used in wireless star networks to avoid collisions by listening before sending data.
Combining Networks
Repeater A device used to regenerate or replicate a signal to extend the range of a network.
Bridge A device that connects network two network segments together and unlike a repeater, a bridge looks at the destination the data is intended for and will not broadcast it across if the destination isn't on te adjacent network segment
Switch A smart network device that connects multiple devices and forwards data to the appropriate device based on MAC addresses
Router Routes data to the appropriate destinations based on IP addresses, essential to the functioning of the internet
Gateway The entrance to the internet on a network, a LAN could have multiple routers but one gateway connecting them to the internet
Client/Server Model
P2P: Peer-to-peer the peer-to-peer model involves processes that provide service to and receive service from client to client, examples are multiple player shooter games, instant messaging and file sharing sites
Distributed System software units that execute as processes on different computers, such as information retrieval systems, company-wide accounting and inventory systems
Cloud Computing Delivery of computing services over the internet, enabling scalable and flexible resources as a service.
Internet Addressing
Tier 1 large ISPs that consist of very high-speed, high-capacity, international WANs
Tier 2 Internet service providers that peer with Tier 1 that tend to be more regional in scope and less potent in their capabilities
Tier 3 Local internet service providers that act as the access to the internet for the vast majority of individuals
IP Address

DNS Domain Name System, translates IP addresses into readable webiste names (ex. 172.217.164.110 = www.google.com)
TLD: Top-level Domain The highest level in the hierarchical Domain Name System of the internet, such as .com, .org
Country-code TLDs regional top-level domains, such as .au ( Australia ), .ru ( Russian ), and .ca ( Canada )
Internet Protocols
FTP File Transfer Protocol, used to transfer and share files over the internet
HTTP HyperText Transfer Protocol, the foundation for websites on the internet
SMTP Simple Mail Transfer Protocol, defines a way that two computers on the network may interact when transmitting an email message from one host to the other
World Wide Web
URL Uniform Resource Locator, contains the information needed by a browser to contact the proper server and request the desired document
Client-side activities Processes and interactions that happen on the user's computer or browser when accessing a website.
Server-side activities Processes handled by the web server such as handling requests, processing data, and managing databases.
Internet Four-layer Software
Application Layer the layer we most commonly interact with responsible for HTTP protocol
Transport Layer known as the connection layer, handles all TCP/IP data so nothing is lost in transit
Network Layer layer responsible for routing and forwarding packets
Link Layer responsible for data transfer between devices on the same network
Cybersecurity
Malware software or code intended to cause damage or inflitrate a system
Virus A type of malware that replicates by inserting itself into other programs or files.
Worm A type of malware that self-replicates and spreads across computers, often through networks.
Trojan Horse A type of malware disguised as legitimate software that, when executed, grants unauthorized access to the system.
Spyware Malware that secretly monitors and collects information about users.
Phishing Fraudulent attempts to obtain sensitive information by pretending to be a trustworthy entity in communication.
DDoS Attack Distributed Denial of Service attack, an attempt to make a network resource unavailable by overwhelming it with traffic.
Protection
Firewall A network security system that monitors and controls incoming and outgoing network traffic based on security rules.
Spam Filter Software that identifies and blocks unwanted emails, typically spam.
Proxy Server A server that acts as an intermediary between a user's device and the internet, often used for privacy or security.
Antivirus Software Software designed to detect, prevent, and remove malware from computers and networks.
Cryptography The practice of secure communication through encoding information, making it readable only to intended recipients.
Public Key Encryption A cryptographic method that uses a pair of keys a public key for encryption and a private key for decryption.
